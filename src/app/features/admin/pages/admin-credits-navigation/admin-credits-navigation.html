Asignar Créditos

Permite asignar paquetes completos de créditos a usuarios. Los créditos se activarán cuando el usuario haga su primera reserva.

Descontar Créditos

Permite descontar créditos como penalización. El descuento se hará priorizando el paquete en uso y aparecerá en el historial del usuario.
